# 🚀 30 Days of Flutter Challenge

Selamat datang di *repository* perjalanan 30 hari saya belajar Flutter dari nol! 
Tujuan dari tantangan ini adalah untuk membangun konsistensi *coding* harian, memahami fundamental Flutter, dan menghasilkan 4 *mini-project* yang layak dijadikan portofolio.

## 🎯 Daftar Proyek
Sepanjang 30 hari, saya akan membangun 4 aplikasi dengan tingkat kesulitan yang terus meningkat:
1. **🌙 Midnight Lines (Day 1-7)** - Aplikasi jurnal malam dan pencatat *mood* dengan tema *dark aesthetics*.
2. **🛠️ Katalog Tutorial (Day 8-14)** - Aplikasi daftar kerajinan tangan dengan navigasi dan *passing data*.
3. **⚽ Papan Skor (Day 15-21)** - Aplikasi pencatat skor pertandingan dengan manajemen *state* dan *timer*.
4. **🌐 Daily Quotes (Day 22-30)** - Aplikasi kutipan harian yang terintegrasi dengan REST API publik dan penyimpanan lokal.
